2026.01.25 SrtEdit 9.1 首发
　　本次有如下几个方面的重大升级，需重新安装：
　　一、底层结构：
　　　１、升级了SrtEdit自定义Undo/ReDo堆栈数据结构及其相关函数，以支持新增的SSA/ASS内嵌字体、图片附件及LRC标识标签等新增数据的撤销与还原操作；
　　　２、升级了SrtEdit自定义剪贴板数据结构及其相关函数，以支持SSA/ASS字幕在复制／粘贴操作中自动适应不同的参考分辨率；
　　　３、重新编译了mpc-hc,将以前SrtEdit与mpc-hc之间用剪贴板传递字幕及视频相关信息改为用WM_COPYDATA消息传递。
　　二、功能增加与改进：
　　　１、支持SSA/ASS内嵌字幕和图片，新增了内嵌附件的附加、提取及删除和使用（仅字体）。附加的内嵌字体可随字幕文件复制到任何环境，且与系统安装的字体使用方法一样(附加字体将出现在样式等编辑器字体列表最后)；
　　　２、全面支持LRC字幕，新增了与之配套的标识标签编辑、预览风格编辑等操作。LRC字幕与SrtEdit所编辑的其他格式字幕编辑方式一致。只是在保存LRC格式字幕时，会自动删除除{\K}以外的风格覆盖代码，并根据每条字幕的显示、消隐时间自动生成空白字幕；自动将{\K}指令转换成为LRC词级时间标签，以实现与SRT等同等效果的字幕显示、消隐及卡拉OK效果；打开LRC字幕时会作相反的转换；　　　　　　　　　　　　　
　　　３、改进了剪贴板操作：复制时只将所选字幕使用的样式写入剪贴板（以前是将当前字幕文件所有样式写入）；粘贴时将对比剪贴板上的样式与当前字幕文件已存在的样式，根据对比结果作如下处理：如果二者完全相同，则不增加新样式，粘贴后的对话、文本也不用作任何改变；如果样式名相同而参数不同，则修改样式名后增加到当前字幕文件，粘贴后对应的对话、文本中的样式名也会随之改变；如果样式名不同而参数相同，则不增加新样式，将粘贴后对应的对话、文本中的样式名改成当前文件对应的样式名。这样既保证了粘贴后的效果与复制前完全一样，又不会增加任何重复的样式。
　　三、其他改进与BUG修正
　　　１、ASS模板增加了分别设定中外文层次参数。使双条双语字幕中外文可使用不同的层次，可分别调整二者显示位置；
　　　２、播放影片拖动进度条可快速浏览音视频，演唱进度及播放影片校准时间也可拖动显示、消隐时间输入框进度条控制音视频播放；
　　　３、修复了自 9.0.2025.1125 版引入的因显示、消隐时间截取函数使用了正则表达式引起效率严重下降，造成双语合成等功能极为卡顿的BUG。
　　　４、使用了消息机制专递字幕、影片文件信息，杜绝了打开影片时可能出现的剪贴板拒绝读写的错误及在非简体中文版下字幕文件名含GBK简体中文字符可能无法加载字幕的BUG.
　　四、其他更新
　　　１、更新 MPC-HC 2.4.2 - Build for SrtEdit 9.0 至 MPC-HC 2.4.2 - Build for SrtEdit 9.1(不与 SrtEdit 9.0 兼容)；
　　　２、更新 LAV 滤波器至 0.80-14-gb6959 版本；
　　　３、更新 MPC视频渲染器至0.9.19.2490版本。

2025.12.18
　　取消了对当前SRT、SSA文件直接应用ASS模板，以防止实际格式与文件名不符。但在批量转换操作中，允许SRT或SSA文件应用ASS模版（会强制转换成为ASS）。
　　在“查看”－“选项”-“文件”的文件页中，增加了新建字幕文件默认格式（SRT/SSA/ASS）选项；在脚本设置中，增加了新建ASS/SSA字幕默认分辨率的设置（SRT的分辨率始终为384x288）；
　　取消了在边框和阴影参数随分辨率改变，因实际效果显示这二者数值不受分辨率影响（脚本设置中要取消“等比例缩放边框和阴影”选项）。
　　复制字幕能保存分辨率信息，粘贴字幕时能根据当前字幕的分辨率与将复制的字幕的分辨率自动换算样式及风格覆盖代码的相关参数。从而使得打开了不同格式、不同分辨率的字幕的多个 SrtEdit 之间可以任意相互复制、粘贴而显示效果不再受分辨率的影响。至此 SrtEdit 已将分辨率问题统一，无论是在当前文件中改变分辨率，还是不同分辨率的自定义样式入库、出库，直至字幕的复制、粘贴都能完美地保持实际显示效果不变。
　　允许对３D字幕进行多行整条字幕进行风格覆盖代码编辑。修正了对话行编辑框等对所选部分行编辑时忽略了３Ｄ字幕另一条字幕同步处理的问题。
　　再次优化了批量转换及操作，允许在文件的批量处理流程中取消。优化了界面及运行反馈信息。

2025.12.10(功能完善)
　　１、进一步完善时间轴录入操作：播放影片时，在2025.11.25版（按鼠标左键暂停/播放、按右键获取当前播放时刻；按住 CTRL 键，按鼠标左键向前跳转、按右键向后跳转）的基础上，增加了按住 Shift 键，按鼠标左键跳转到前一条字幕、按右键跳转到后一条字幕。且空时间轴不再影响播放（以前会跳至片头），这样在录入时间轴前，先将需录入的时间轴清空，就可边录入边观看结果。
　　２、规范了样式编辑器，使其只能借用某条字幕对样式本身进行编辑预览，而不能改变任何一条字幕的样式名。规则是：对于有字幕使用的样式，将自动跳转到该条字幕进行样式编辑预览；对于新建样式或未使用的样式，可借用本条字幕对其进行样式编辑预览，退出样式编辑窗口后还原本条字幕样式，不会对其有任何影响。
　　３、完善了对话行字段编辑：在搜索对话行字段的基础上，增加了替换对话行字段选项（在对话编辑框左侧也增加了查找替换按钮）。从而可以在搜索的同时用其他样式替换字幕正在使用的样式，而不必先搜索完成后再在对话编辑框中选择其他样式。
　　４、完善了 ASS 模版的管理。规范了其生成、修改、应用及导入、导出流程。且在应用ASS模版时，如果选择的是不删除原字幕风格覆盖代码，则会根据当前字幕的参考分辨率自动修正所有风格覆盖代码有关位置、大小的参数，以保证显示效果不变。
　　５、加强了自定义样式的出库、入库及导入、导出管理。完善了自定义样式，增加了分辨率属性。使得不同分辨率下的字幕定义的自定义样式能够相互通用，不再会造成高分辨率下的样式应用到低分辨率字幕时文字变大或者相反的现象。
　　６、进一步优化了批量转换及操作，修正了批量处理进行时，频繁激活主窗口，以至于影响其他应用窗口不能正常输入以及导出批量处理报告保存对话框时内存出错的问题。
　　７、修正了上一版引起的所选区域时间轴缩放不能全选、以及2025.726版引起的在播放影片标签切换到文本编辑标签时文本编辑框不能进入等问题。优化了一些算法及修正了一些BUG。

2024.12.03（重要更新）
　　增强和完善了批量文件操作处理功能。增强了操作命令批量编排，与以前同一操作只能运行一次不同，现在已允许多次运行。主要有以下改进：
　　１、对源文件列表中的字幕文件进行批量输入输出自动转换处理，包括字幕格式转换、字符集编码的转换等（旧版已有）。
　　２、对源文件列表中的每个字幕文件进行批量操作，相当于编程。当源文件列表为空时，可对当前正在编辑的字幕文件进行批量操作。否则将对源文件列表中的字幕文件进行操作处理，而当前正在编辑的字幕文件不受影响（旧版不能对当前文件批量操作，且批量操作同一操作命令只能执行一次）。
　　３、列表可将一些较复杂的多步操作才能实现的功能保存为CmdList文件导入执行，类似于 bat 批命令（旧版无）。
　　４、开放了 SrtEdit 能单独运行的全部操作命令，以前由于一些操作需用对话框设置参数而不能批量自动执行，现在可在编排界面直接打开对话框预先设定默认参数（旧版不能运行需对话框交互的操作命令）。
　　５、修正了相关的bug。

2024.11.11 SrtEdit 8.1（重大更新）
　　１、增加了加载DVD/BD光盘功能，可为当前字幕直接加载光盘影片。为此，mpc-hc 升级到 build for SrtEdit 8.1，以便能够自动将当前字幕加载到光盘文件播放。 
　　２、恢复并改进了字幕预览功能。在缺少字幕相应视频文件时，可自动调用用户指定的预览视频文件或 SrtEdit 8.1 自动生成的单色背景的默认预览视频文件。用户还可自行设定默认预览视频的时长(默认3小时，可设置为半小时至八小时)、分辨率、背景色等。
　　３、重写了双语字幕自动合成及人工处理的相关程序，改进了算法，提高了双语字幕合成的准确度，纠正了以前可能丢失字幕的现象。
　　４、优化了主界面，全面梳理了各编辑框快捷键，突出其焦点状态。各编辑框快捷键被限制在本区域，因此各个编辑框可以使用相同的快捷键而不再冲突。
　　５、根据用户的反馈和建议修正了许多BUG和功能改进。例如不再把一些全角符号（如音乐符号、不能简单转半角的如全角希腊字幕等）划归中文字符，以防止把外文字幕识别为中文字幕。
　　由于 SerEdit 8.1 作了一些结构上的改动，且使用 mpc-hc build for SrtEdit 8.1。建议接受安装程序删除旧版建议，完全安装。
